# 📸 Frouane Photographie - Portfolio & E-Commerce

Site web professionnel de photographie maritime par **Franck Rouane**, photographe spécialisé dans les paysages méditerranéens, les calanques et la Côte d'Azur.

🌐 **Site en ligne :** [frouanephotographie.com](https://frouanephotographie.com)

---

## ✨ Caractéristiques

### 🎨 Portfolio Immersif
- **6 collections** de photographies maritimes
- Galerie interactive avec **effet hero zoom**
- Animations fluides au scroll
- Lightbox haute qualité
- Carte interactive des prises de vue (Leaflet)
- Design élégant noir/blanc/or

### 🌍 Sensibilisation Écologique
- Contenu enrichi sur les espèces protégées
- Menaces environnementales documentées
- Conseils de respect des lieux
- Intégration naturelle dans le storytelling

### 🛒 E-Commerce Complet
- Panier persistant avec calcul automatique
- Wishlist/Favoris avec icônes cœur
- Système de codes promo (-10%)
- **Popup newsletter exit-intent** avec génération code promo
- Formats multiples (30x40cm à 100x150cm)
- Gestion éditions limitées

### 📊 Analytics & SEO
- Google Analytics 4 intégré
- Google Search Console configuré
- Tracking événements e-commerce
- Sitemap.xml automatique
- Meta tags optimisés
- Schema.org markup
